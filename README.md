## WalletLibRLP

The WalletLibRLP library is an implementation of tools related to Ethereum transactions encoding and decoding.
## Requirements

* macOS 12.6
* XCode Version 14.2

## Installation

Using `CocoaPods`. 

Clone or download repo, add the following line to your `Podfile`

```ruby
# platform :ios, '10.0'

target 'YourTargetName' do
  use_frameworks!
  
  pod 'WalletLibRLP', :path => 'path/to/WalletLibRLP' 
end
```
